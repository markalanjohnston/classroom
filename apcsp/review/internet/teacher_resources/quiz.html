AP Computer Science Principles

Internet Basics Review Quiz

Name:                              Class Period:           
Directions: Answer each question using complete thoughts when needed. This quiz uses short answer and fill-in-the-blank questions. There are no multiple-choice questions.

1. Explain the difference between the Internet and the World Wide Web.

2. What is the main job of DNS?

3. When data is sent over the Internet, it is broken into smaller pieces called ______________________________.

4. Name two pieces of information, or metadata, that are included in each packet.

5. Do packets always travel along the same path from sender to receiver? Explain.

6. What is fault tolerance, and how does the Internet achieve it?

7. Which protocol ensures that packets are delivered in the correct order and resends missing packets?

8. What is the difference between HTTP and HTTPS?

9. Define bandwidth and latency. You must define both.

10. A video takes a long time to start playing, but once it starts, it plays smoothly in high quality. Is this a problem with bandwidth or latency? Explain.
