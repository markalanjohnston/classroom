AP CSP Review Lesson: Reading Pseudocode + Mixed MCQ Practice

Today’s Goal: Learn to read the AP CSP pseudocode reference sheet confidently, then apply that skill on a mixed practice set of real questions from the College Board’s 2021 AP CSP Practice Exam.

Today’s Agenda

TimeActivity
0–5 min Opener: Reference Sheet preview — confirming what the College Board gives you on test day and how to use it.
5–25 min Guided Walkthrough: We will work through the Reading Pseudocode review page together — assignment, operators, IF / IF-ELSE, REPEAT, FOR EACH, lists, procedures, and tracing.
25–40 min Partner Tracing Practice: With a partner, trace two short pseudocode snippets on paper. Predict the final values of every variable before checking against the answer.
40–75 min Mixed MCQ Practice: Work through the 20-question mixed MCQ practice set individually. Real questions from the released 2021 AP CSP Practice Exam. Goal pace: ~1m 45s per question (the real-exam target).
75–85 min Class Review: We will go over the questions students missed most often and discuss the reasoning behind the right answers.
85–90 min Exit Ticket: Quick reflection before leaving.

Key Topics We Are Reviewing

  • The AP Reference Sheet — what it is, where you find it on the exam, how to use it
  • Assignment (a ← expression) vs. equality (a = b)
  • Arithmetic operators, including integer division and modulus
  • Comparison and Boolean operators (AND, OR, NOT)
  • Selection: IF and IF / ELSE
  • Iteration: REPEAT n TIMES, REPEAT UNTIL, and FOR EACH
  • Lists — indexing, length, appending, inserting, removing
  • Procedures and RETURN
  • Robot commands (MOVE_FORWARD, ROTATE_LEFT, ROTATE_RIGHT, CAN_MOVE)
  • How to trace code — tracking variables in a table
  • Common AP exam misconceptions

Partner Tracing Requirements

With your partner, complete two pseudocode tracing exercises (provided in class). For each one you should:

  • Build a trace table that lists every variable in the snippet.
  • Update the table line-by-line as you work through the code.
  • Predict the final value of every variable and any output before checking the answer.
  • If you and your partner disagree, slow down and re-trace the line you disagree on.

Practice Set Information

You will work through a 20-question mixed MCQ set drawn from the College Board’s 2021 AP CSP Practice Exam. Each question has instant feedback with the explanation from the official answer key.

Treat the score tracker at the top of the page as your data: pay attention to which topic tags you miss most. That is your study list.

Pacing target: roughly 1 minute 45 seconds per question. This matches the real AP exam timing.

Class Expectations

  • Use the reference sheet during the walkthrough — the goal is fluency with it, not memorization.
  • Trace carefully with your partner. Speed is not the goal yet; accuracy is.
  • Work the practice set honestly — the feedback is only useful if your answers are your own.
  • Pay attention to topic tags on the practice questions. They tell you where to study.
  • Participate in the class review — the misses are the most valuable part of the lesson.
