Directions. With your partner, trace each snippet line-by-line using the table provided. Work on paper before checking the answer.

  1. Start at the top of the snippet. Write the value of each variable after the line runs.
  2. Add a new row to the table every time a variable changes (or once per loop pass, whichever is clearer).
  3. Predict the final output before raising your hand for the answer.
  4. If you and your partner disagree on a value, slow down and re-trace the line you disagree on. Do not move forward until you both agree.

Reminders: ← stores the right side into the variable on the left. AP CSP lists are 1-indexed — aList[1] is the first element. MOD returns the remainder, not the quotient.

Snippet 1 — Loop with an IF / ELSE inside

a ← 1
b ← 10
REPEAT 4 TIMES
{
    IF (a MOD 2 = 1)
    {

    }
