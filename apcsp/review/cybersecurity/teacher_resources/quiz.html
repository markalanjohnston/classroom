Cybersecurity & Encryption Review Quiz

Name:                              Class Period:           
Directions: Answer each question using complete thoughts when needed. This quiz uses short answer, fill-in-the-blank, and matching questions. There are no multiple-choice questions.

1. Define encryption and decryption. You must define both.

2. Explain the main difference between symmetric and asymmetric (public-key) encryption.

3. In public-key encryption, Alice wants to send Bob a confidential message. Whose key does Alice use to encrypt the message, and whose key does Bob use to decrypt it? Be specific about both the person and whether it is the public or private key.

4. What is the main advantage of public-key (asymmetric) encryption compared to symmetric encryption?

5. What is the purpose of a digital certificate, and what kind of organization issues one?

6. A website is using HTTPS (the padlock icon is showing in the browser). Does this mean the website itself is trustworthy and safe? Explain.

7. What is Multi-Factor Authentication (MFA)? Name the three categories of authentication factors and give one example of each.

8. A website requires a password and the answer to a security question (such as "What was the name of your first pet?"). Is this Multi-Factor Authentication? Explain why or why not.

9. Match each scenario to the correct threat term. Write your answer on the blank to the right of each scenario.

Word bank: phishing  ·  keylogger  ·  rogue access point  ·  virus  ·  worm
a.
A user receives an email that appears to be from their bank, asking them to click a link and "verify" their login.
b.
Hidden software on a laptop secretly records every keystroke a user types, including passwords.
c.
An attacker sets up an unauthorized Wi-Fi hotspot in a coffee shop and names it "FreeCoffeeWiFi" to intercept users' traffic.
d.
A piece of malware spreads automatically across networked computers without any user action and without needing a host file.
e.
A piece of malware attaches itself to a program file and only spreads when the user runs that program.

10. What is PII? Why is the combination of ZIP code, date of birth, and gender often considered a privacy risk, even though each piece by itself may seem harmless?
