The two tracks (what actually happens in each room)

TRACK G Getting Acquainted — Cafeteria, ~70 min, no computers

Led by: P-TECH senior(s). Mr. Johnston floats. No tables needed.

  1. Icebreaker ball toss (~20 min). Form a circle of 10–12 (more circles if large). Toss a ball; catcher says their name + answers the question under their right thumb, then tosses on. Keep it fast; "pass" is OK.
  2. Quick team game (~15 min). Pick one, no writing: silent birthday line-up · "cross the circle if…" · Two Truths & a Lie in clusters of 4.
  3. Harold the Robot (~25 min). 10-min whole-group demo: an adult is "Harold," a robot that obeys only simple literal commands ("move hand left," "close hand," "lift") while students direct it to stack cups/books into a tower. Then break into groups of 4–6, each with their own "robot."
  4. Buffer (~10 min). Another ball-toss round, or "write your name in binary" using the binary cards.
Debrief (2 min): "Why was Harold so hard to boss around?" → Computers do exactly what you say, nothing more. Precise instructions = code. That's what you'll do on the computers next.

TRACK C Coding — Computer Lab (26 machines), ~70 min

Led by: Mr. Johnston + a senior/AP floor helper. Pairs if a group is over 26.

  1. Log in & settle (~5 min). Hand each student a login slip → go to academy.cs.cmu.edu → click Login → enter the username & password → open the Hour of Code lesson.
  2. Teacher demo (~10 min). On the projector, run the first activity — draw a shape, change a color, press Run. Show that the code is just instructions.
  3. Work time (~45 min). Students work through Hour of Code: Drawing with Code → Using the Mouse → Changing Behavior. Helpers walk the rows — don't type for them; ask "what do you want to happen?"
  4. Share & log off (~10 min). Show your drawing to a neighbor; finishers get free-draw time; log off.
Debrief (2 min): "You just wrote real Python that draws on a screen." Tease Wednesday: we'll go further — games, robots, maybe a real circuit.
