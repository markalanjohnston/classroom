Carnegie Mellon University Computer Science Academy

CS1: Introduction to Programming with Graphics

Course Overview

CMU CS Academy provides a comprehensive introduction to computer science and programming through interactive graphics. Students learn fundamental programming concepts including shapes, functions, conditionals, loops, lists, and game development using Python. The curriculum emphasizes hands-on learning and creative problem-solving through visual programming projects.
