FOLLOW ALONG

10 minutes

Log into CMU Academy and open Unit 4.1 Notes - We will go through this together

Using "else"

Let's start with this example for review:

Label("Using 'else' demo #2", 200, 20, size=20, bold=True)
Label('Cyan on left, gold on right', 200, 40, size=16)
Line(200, 50, 200, 400, fill='lightGray')

def onMousePress(x, y):
    if (x < 200):
        Circle(x, y, 30, fill='cyan')
    else:
        Circle(x, y, 30, fill='gold')

Run the code and see how you get cyan circles on the left, and gold circles on the right. Look closely at line 8 (the else statement). We use else after an if to run code if the previous test was False. In this example, since the previous if tested if the click was on the left, the else tests if the click was on the right.

Using "elif"

Right after we use if, we can use elif to add another test that will only be tested if the first test was False. For example:

Label('Using "elif" demo #1', 200, 20, size=20, bold=True)
Label('Cyan on left, black in middle, gold on right', 200, 40, size=16)
Line(100, 50, 100, 400, fill='lightGray')
Line(300, 50, 300, 400, fill='lightGray')

def onMousePress(x, y):
    if (x < 100):
        Circle(x, y, 30, fill='cyan')
    elif (x < 300):
        Circle(x, y, 30, fill=None, border='black')
    else:
        Circle(x, y, 30, fill='gold')

Run the code and see how you get cyan circles on the left, black rings in the middle, and gold circles on the right. Look closely at line 9 (the elif statement). This test only happens if the first test is False. So we can only get black rings if we do not get cyan circles.

elif is a short form of saying "else if". This is similar to what we do in real life if we have more than 2 choices. For example, since there are more than 2 types of weather, we might want to program something like this:

if (isSnowing == True):
    bring('jacket')
elif (isRaining == True):
    bring('umbrella')
else:
    wear('sunglasses')

Checkpoint 1

3 minutes

What happens in the previous example if we used if instead of elif?

USE ABCD CARDS

  • The cyan circles are drawn with a black ring. (CORRECT!)
  • Only black rings are drawn.
  • Both the cyan circles and the gold circles are drawn with black rings.

Kinesthetic Conditional Activity

10 minutes

Explain: “We’re going to act like a computer program.” Students stand.

Prompt 1 (Birth Months)

  • IF you were born before March → move to the left side of the room.
  • ELIF you were born in April, May, or June → move to the middle.
  • ELSE → move to the right side.
Ask:
  • “Which group was the IF group?”
  • “Which group was ELIF?”
  • “Who ended up in ELSE?”
Explain:
“The computer checks conditions in order.”

Prompt 2 (Clothing Example)

Students reset.

  • IF you are wearing black shoes → move left
  • ELIF you are wearing white shoes → move center
  • ELSE → move right
Ask:
“Why didn't the computer check the second condition for some of you?”

Goal: reinforce conditional order.

FOLLOW ALONG PART 2

10 minutes

Using more than one "elif"

We can use as many elif's as we need. Each elif is only tested if none of the previous tests were True. For example:

Label("Using 'elif' demo #2", 200, 20, size=20, bold=True)
Label('Now with 4 colors!', 200, 40, size=16)
Line(100, 50, 100, 400, fill='lightGray')
Line(200, 50, 200, 400, fill='lightGray')
Line(300, 50, 300, 400, fill='lightGray')

def onMousePress(x, y):
    if (x < 100):
        Circle(x, y, 30, fill='cyan')
    elif (x < 200):
        Circle(x, y, 30, fill='plum')
    elif (x < 300):
        Circle(x, y, 30, fill='teal')
    else:
        Circle(x, y, 30, fill='gold')

Another example

Let's look at one more example:

Label("Using 'elif' demo #3", 200, 20, size=20, bold=True)
Label('Click in each of the 4 squares', 200, 40, size=16)
Line(200, 50, 200, 400, fill='lightGray')
Line(0, 200, 400, 200, fill='lightGray')

def onMousePress(x, y):
    if (x < 200):
        Circle(x, y, 30, fill='cyan')
    elif (y < 200):
        Circle(x, y, 30, fill=None, border='black')
Question:

Why do we only get black rings in the top-right and not in the top-left?

Answer:

The if tests if the click is on the left, and we can only get to the elif if that test was False, so only if the click was on the right. Then the elif tests if the click is on the top, so that test only passes if the click is both on the right and on the top.

Checkpoint 2

In the previous example, which of the four squares changes behavior if we used if instead of elif? (Try it first!)

  • The bottom-right square (black rings are drawn).
  • Nothing changes.
  • The top-left square (the cyan circles are drawn with a black ring). (CORRECT!)
  • The top-right square (black rings are not drawn).
Think–Pair–Share

Why does the order of IF → ELIF → ELSE matter in a program?

ABCD Card Formative Assessment Part 1

5 minutes
Display question:

Question 1

What does elif mean?

  • A - Run code when the condition is true
  • B - Another condition checked only if previous ones are false
  • C - A loop
  • D - A variable

Correct answer: B

Question 2

What happens if we replace elif with if?

  • A - Nothing changes
  • B - Multiple conditions may run
  • C - The program crashes
  • D - The program runs faster

Correct answer: B

Short Group Activity

5–7 minutes

This works well right before the Stoplight coding activity.

Activity: Human Algorithm

Put students into groups of 3–4.

Give each group this scenario:

Create a conditional algorithm for deciding what to wear.

Example structure:
IF ______
ELIF ______
ELSE ______
Example responses students might produce:
IF temperature < 40 → wear a jacket
ELIF temperature < 70 → wear a hoodie
ELSE → wear a t-shirt

Groups share one example.

Quickly connect:

"You just wrote a real-world algorithm using conditionals."

Unit 4.1.2 “Stoplight” Exercise:

Red light! Green light!
Press the mouse to change the stoplight. Notice how after three mouse presses, it ends up at a red light again.

Students are given:
app.background = 'deepSkyBlue'

Line(0, 120, 400, 120, fill='darkGrey', lineWidth=90)
Rect(85, 20, 230, 360, fill=rgb(30, 30, 30))

redLight = Circle(200, 90, 50, fill='red')
yellowLight = Circle(200, 200, 50, fill='gold', opacity=30)
greenLight = Circle(200, 310, 50, fill='limeGreen', opacity=30)

def onMousePress(mouseX, mouseY):
    # Change the opacity of the lights so that the stoplight changes color when
    # the mouse is pressed!
    ### Place Your Code Here ###
    pass
They must add:
if (redLight.opacity == 100):
    redLight.opacity = 30
    greenLight.opacity = 100
elif (greenLight.opacity == 100):
    greenLight.opacity = 30
    yellowLight.opacity = 100
else:
    yellowLight.opacity = 30
    redLight.opacity = 100

Circulate giving feedback.
