Teacher Lesson Plan

Lesson 306: Unit 1 Exercises – Catch-Up and Support Day

Lesson Overview

Course: Fundamentals of Computer Science
Lesson Code: 306
Lesson Title: Unit 1 Exercises – Catch-Up and Support Day
Class Length: 90 minutes
Curriculum Focus: All Unit 1 material prior to the Creative Mini-Task

This lesson is a structured work and support day. Students use the time to complete unfinished CMU CS Academy Unit 1 lessons and exercises, review challenging concepts, and receive targeted assistance. The goal is for all students to reach readiness for the upcoming creative task by strengthening foundational drawing and programming skills.

Objectives

Students will:

  • Complete any unfinished CMU CS Academy Unit 1 lessons and exercises.
  • Review and correct misunderstandings about shapes, coordinates, colors, and optional parameters.
  • Demonstrate improved independence in debugging and problem solving.
  • Prepare to begin the Unit 1 Creative Mini-Task with confidence.

Materials & Setup

  • Student computers with internet access
  • CMU CS Academy login credentials
  • Student notes from previous lessons
  • Teacher access to the CMU CS Academy teacher dashboard

Teacher Preparation:

  • Review class progress data to identify students needing the most support
  • Prepare a short list of common trouble spots to review at the start of class
  • Arrange seating to support pair programming or small-group help as needed

Agenda (90 Minutes)

Time Activity
0 – 10 minClass Reset and Goal Setting
10 – 20 minMini-Review of Key Unit 1 Concepts
20 – 80 minIndependent Work and Teacher Support
80 – 87 minProgress Check and Reflection
87 – 90 minClosure and Homework

Teacher Moves (Scripted Talking Points + Demo Steps)

Opening (10 minutes)

Talking Points

"Today is a support and catch-up day. The goal is simple: get everyone fully ready for the Unit 1 Creative Mini-Task."

"You should leave today with all Unit 1 lessons and exercises completed or very close to finished."

"This is also your chance to ask questions, fix errors, and make sure you really understand what you've been learning."

Explain expectations:

  • Work independently
  • Ask for help when stuck
  • Use problem-solving strategies before calling the teacher

Mini-Review (10 minutes)

Briefly review the most important Unit 1 ideas:

  • How coordinates work on the canvas
  • The difference between shapes using radius vs width/height
  • Using fill colors and opacity
  • Optional parameters
  • Importance of code order and layering

Quick demonstration as needed:

  • Show how to use the Inspector tool
  • Model reading an error message
  • Demonstrate fixing a common mistake

Independent Work Time (60 minutes)

Teacher role:

  • Circulate the room
  • Monitor student progress
  • Provide one-on-one help
  • Pull small groups for targeted re-teaching

Suggested teacher actions:

  • Sit with students who are behind
  • Help students interpret autograder feedback
  • Encourage pair programming for difficult problems

Student Tasks

In-Class Tasks

Students will:

  • Log into CMU CS Academy
  • Open Unit 1 lessons and exercises
  • Work on any activities not yet completed
  • Revise incorrect solutions based on feedback
  • Ask for help when needed

Priority order for students:

  1. Finish required lessons
  2. Complete assigned exercises
  3. Fix any previously incorrect work

Homework

Any remaining unfinished Unit 1 lessons or exercises must be completed outside of class so students are prepared to begin the Creative Mini-Task.

Checks for Understanding (Formative)

Throughout the period, ask students questions such as:

  • "Explain what this line of code is doing."
  • "Why did this shape appear on top of that one?"
  • "How did you decide which shape command to use?"
  • "What does this error message mean?"

End-of-class quick reflection:

Each student answers:

  • What did you finish today?
  • What concept is still confusing?
  • Are you ready for the creative task?

Common Misconceptions & Fixes

Issue How to Address
Shapes in wrong location Review coordinate system and parameter order
Colors not applying Check spelling and quotes around color names
Code runs but drawing looks wrong Use Inspector tool to compare values
Students stuck on autograder Encourage reading feedback carefully
Confusion about parameters Revisit required vs optional parameters

Differentiation & Accommodations

For students behind

  • Provide focused help on essential exercises only
  • Allow pair programming
  • Offer step-by-step hints

For advanced students

  • Encourage completing optional challenges
  • Ask them to assist peers
  • Begin planning ideas for the creative task

General supports

  • Small group reteaching
  • Visual reminders of commands
  • Flexible pacing within the class period

Assessment Notes

This lesson is not graded directly.

Assessment is based on:

  • Completion of CMU CS Academy Unit 1 activities
  • Demonstrated readiness for the upcoming Creative Mini-Task
  • Informal teacher observations of understanding

Closure

"Today's work sets you up for success on the Unit 1 Creative Mini-Task. If you finished everything, great. If not, make sure to complete the rest as homework."

"Coming prepared next class will make the creative project much easier and more enjoyable."
