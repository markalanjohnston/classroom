FLEX DAY #1 - Teacher Resource

Reteach / Catch-Up / Practice Suggestions

Purpose of This Flex Day

This is the first scheduled flex day of the semester. At this point, students have completed:

  • Unit 1.1 – Basics: Drawing basic shapes (Rect, Circle, Oval, Line)
  • Unit 1.2 – Colors: Using fill, opacity, and RGB values
  • Unit 1.3 – Shapes: Additional shapes (Star, Polygon, Label) and coordinate placement

Use this day strategically to ensure all students have a solid foundation before moving into functions and interactivity in Unit 2.

Suggested Uses for This Class Period

Option 1: Catch-Up Day

Best for: Classes where many students have incomplete exercises

Structure:

  • 5 min: Review expectations and check CMU CS Academy progress
  • 75 min: Independent work time on incomplete Unit 1 exercises
  • 10 min: Progress check and closure

Teacher Actions:

  • Pull up CMU CS Academy teacher dashboard
  • Identify students with incomplete exercises
  • Circulate and provide one-on-one support
  • Prioritize students who are furthest behind

Option 2: Reteach Struggling Concepts

Best for: Classes where specific concepts caused widespread confusion

Common Struggle Areas:

  • Coordinates: Students confusing x/y or not understanding (0,0) placement
  • RGB Values: Confusion about 0-255 range or mixing colors
  • Shape Parameters: Using wrong number of arguments or mixing radius with width/height
  • Opacity: Not understanding 0 (invisible) vs 100 (solid)
  • Code Order: Not recognizing that shapes layer from top to bottom

Mini-Lesson Structure (20-30 min):

  • Re-explain the concept with visual examples
  • Model common mistakes and how to fix them
  • Do 2-3 guided practice problems together
  • Release students to independent practice

Option 3: Small Group Differentiation

Best for: Classes with varying skill levels

Group 1 – Struggling Students:

  • Teacher-led reteach session
  • Simplified starter code templates
  • Focus on Unit 1.1 and 1.2 only

Group 2 – On-Track Students:

  • Independent catch-up and practice
  • Complete all Unit 1 exercises
  • Peer support encouraged

Group 3 – Advanced Students:

  • Creative challenge: "Recreate a famous painting using only basic shapes"
  • Extension: Explore advanced shape combinations
  • Peer tutoring role for struggling classmates

Option 4: Collaborative Debugging Practice

Best for: Building problem-solving skills

Activity Structure:

  • Provide students with intentionally broken programs (5-7 bugs each)
  • Students work in pairs to find and fix all bugs
  • Common bugs to include:
    • Missing quotes around color names
    • Swapped x and y values
    • RGB values over 255
    • Missing or extra parentheses
    • Shapes drawn in wrong order
  • Debrief as a class: What strategies helped you find bugs?
Timing: 15 min setup + 50 min practice + 15 min debrief

Option 5: Creative Application Task

Best for: Classes where most students are caught up

Mini-Project Ideas:

  • "Design Your Emoji": Create a custom emoji face using shapes
  • "Scene Creator": Draw a simple landscape or room
  • "Logo Design": Create a logo using only shapes and colors
  • "Abstract Art": Make an interesting abstract composition

Requirements:

  • Use at least 5 different shapes
  • Use at least 3 colors (including one RGB custom color)
  • Use opacity on at least one shape
  • Code must be organized and easy to read
Grading: This could serve as an informal formative check or a small completion grade

Data to Collect Today

Use this flex day to gather information about student progress:

  • How many students completed all Unit 1 exercises?
  • Which specific lessons have the most incomplete work?
  • What concepts are students still struggling with?
  • Who needs extra support moving forward?
  • Is the class ready to move into Unit 2, or does pacing need adjustment?

Before Moving to Lesson 306

By the end of today's flex day, students should ideally:

  • Have completed all assigned Unit 1.1, 1.2, and 1.3 exercises
  • Be comfortable with basic shapes, colors, and coordinate placement
  • Understand how to read error messages and debug simple programs
  • Feel confident and ready to move into functions and mouse events

Note: It's okay if a few students still have minor gaps, but the majority should have a solid Unit 1 foundation before starting Unit 2.

Teacher Reflection Questions

After today's flex day, consider:

  • Did this flex day accomplish its purpose?
  • Are students ready to move forward, or do we need another catch-up session?
  • What patterns emerged in student struggles?
  • Should I adjust my pacing or instructional approach for future units?
  • Which students need targeted intervention before Unit 2?
