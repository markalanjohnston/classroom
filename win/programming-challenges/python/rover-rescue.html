Rover Rescue Mission

Welcome to the ultimate Python programming challenge! In this advanced mission, you'll program a rover on an alien planet to collect minerals, neutralize enemies, and survive in a hostile environment. This is the capstone project that combines everything you've learned!

Mission Briefing

You are the sole programmer for a research rover stranded on an alien planet. Your mission: program the rover to autonomously collect minerals, manage its battery, avoid hazards, and neutralize radioactive enemies. The fate of the mission depends on your code!

Mission Outcomes

By completing this mission, you will:

  • Master advanced Python commands for autonomous navigation
  • Implement complex decision-making algorithms
  • Use AI-powered sensors for object detection and tracking
  • Optimize code for efficiency and battery management
  • Create event-driven programs for dynamic responses
  • Combine all VEX VR Python skills into a comprehensive solution

Mission Objectives

Your rover must survive on an alien planet by completing the following objectives:

Collect Minerals

Find and collect minerals scattered across the terrain. Minerals can be:

  • Transported to Base: +5 XP each
  • Consumed for Energy: +2 XP + Full battery recharge
Manage Battery

Your rover's battery constantly drains. You must:

  • Monitor battery levels
  • Use minerals to recharge
  • Use standby mode strategically
Neutralize Enemies

Absorb radiation from alien creatures:

  • Alien Spiders: 5 XP
  • Orange Serpents: 10 XP
  • Blue Serpents: 10 XP
  • Purple Serpents: 15 XP
Level Up

Gain Experience Points (XP) to level up your rover:

  • Higher levels = more storage capacity
  • Higher levels = tougher enemies
  • Strategic XP management is key
Warning: If your rover's battery reaches 0%, the mission fails! Always keep an eye on your battery level and plan your actions carefully.

Key Commands for Rescue Operations

The Rover Rescue mission uses specialized Python commands that go beyond basic movement. Here are the critical commands you'll need:

Advanced Navigation

drivetrain.go_to(object)

Turn and drive the rover to a specified object in one command.

# Go to the nearest mineral drivetrain.go_to(MINERALS) # Go to the nearest enemy drivetrain.go_to(ENEMY) # Go to the base (always works, even if base is not visible) drivetrain.go_to(BASE)

drivetrain.turn_to(object)

Turn the rover to face a specified object without moving forward.

# Turn to face minerals drivetrain.turn_to(MINERALS) # Turn to face the base drivetrain.turn_to(BASE)

drivetrain.drive_to(object)

Drive forward to an object without turning first (rover must already be facing it).

# Drive to base (must be facing base first) drivetrain.drive_to(BASE)

Mineral Management

rover.pickup(MINERALS)

Pick up minerals and store them in the rover's cargo.

# Navigate to minerals and pick them up drivetrain.go_to(MINERALS) rover.pickup(MINERALS)

rover.drop(MINERALS)

Drop minerals from cargo at the current location.

# Go to base and drop minerals for XP drivetrain.go_to(BASE) rover.drop(MINERALS) # +5 XP per mineral

rover.use(MINERALS)

Consume minerals on the ground to recharge battery and gain XP.

# Use minerals for energy (minerals must be on ground, not in cargo) drivetrain.go_to(MINERALS) rover.use(MINERALS) # +2 XP + Full recharge!
Important: Minerals must be on the ground to use them. You cannot use minerals in your cargo!

Enemy Interaction

rover.absorb_radiation(ENEMY)

Absorb radiation from enemies to neutralize them and gain XP.

# Approach and neutralize enemy drivetrain.go_to(ENEMY) rover.absorb_radiation(ENEMY)

Power Management

rover.standby(percent)

Enter standby mode until battery reaches a specified threshold. Conserves battery but increases time counter faster.

# Enter standby until battery reaches 50% rover.standby(50)

AI Sensors

rover.sees(object)

Check if the rover sees an object within its 40-degree field of view (1000mm range).

# Check if minerals are visible if rover.sees(MINERALS): drivetrain.go_to(MINERALS) rover.pickup(MINERALS)

rover.detects(object)

Check if the rover detects an object within its 800mm detection radius (360-degree range).

# Check if enemies are nearby if rover.detects(ENEMY): brain.print("Enemy detected!")

Status Monitoring

Rover Status Commands

# Battery level (0-100) battery_level = rover.battery() # Minerals in storage minerals_count = rover.minerals_stored() # Storage capacity capacity = rover.storage_capacity() # Current rover level level = rover.level() # Current experience points xp = rover.exp() # Check if under attack if rover.under_attack(): brain.print("Under attack!")

Event Handlers

Event-Driven Programming

Create functions that automatically run when certain events occur.

# Define function for when under attack def under_attack(): brain.print("Under attack!") drivetrain.drive_for(REVERSE, 500, MM) # Define function for leveling up def level_up(): brain.print("Leveled up!") # Register the event handlers rover.on_under_attack(under_attack) rover.on_level_up(level_up) wait(0.15, SECONDS) # Allow time to register

Strategy Guide: Breaking Down the Mission

Success in the Rover Rescue mission requires strategic thinking and efficient code. Here's how to approach this challenge:

Phase 1: Early Game (Zones A & B)

Survival Strategy

  • Priority: Collect minerals and build XP
  • Battery Management: Use minerals for energy when battery drops below 30%
  • Enemy Strategy: Focus on Alien Spiders (easiest, 5 XP each)
  • Mineral Strategy: Use minerals for recharge rather than transport initially

Phase 2: Mid Game (Zone C)

Growth Strategy

  • Priority: Level up and increase storage capacity
  • Battery Management: Balance between using and transporting minerals
  • Enemy Strategy: Start tackling Orange Serpents (10 XP)
  • Mineral Strategy: Minerals respawn here - establish a routine

Phase 3: Late Game (Zones D & E)

Advanced Strategy

  • Priority: Maximize XP per action
  • Battery Management: Transport minerals to base when storage is full
  • Enemy Strategy: Target Blue and Purple Serpents (10-15 XP)
  • Mineral Strategy: Fast respawn rate - create efficient loops

Key Strategic Decisions

Decision Tree

When you find a mineral, should you use it or transport it?

  • Use if: Battery < 30% OR not near base
  • Transport if: Battery > 70% AND near base AND storage not full
  • Pick up for later if: Storage not full and might need energy later

Enemy Engagement

When should you fight enemies?

  • Engage if: Low-level enemy AND battery > 50%
  • Avoid if: High-level enemy AND low battery
  • Flee if: Under attack AND battery < 20%

Collecting minerals...") drivetrain.go_to(MINERALS) rover.pickup(MINERALS) drivetrain.go_to(BASE) rover.drop(MINERALS) main()

Tutorial 5: Continuous Operation with Loop

Objective: Create a sustainable collection loop
