Challenge Projects

Put all your skills to the test! These comprehensive challenges combine everything you've learned. Choose your difficulty level and show what you can do!

How It Works: Each challenge has specific objectives and a rubric. Complete as many as you can. These are designed to be challenging - don't worry if you can't complete them all on your first try!

Secret Message Decoder

Advanced
Encoded Message

Decode a secret binary message by reading colored tiles. Blue = 0, Green = 1.

Objectives:
✓ Navigate the grid systematically
✓ Read colors with down eye sensor
✓ Convert binary to decimal
✓ Display the decoded message
Eye Sensor Variables Data Processing Algorithms
Algorithm Hints

Step 1: Create a systematic path (like reading a book - left to right, top to bottom)

Step 2: Store each bit (0 or 1) in a list or string variable

Step 3: Every 8 bits makes one character

Step 4: Convert binary to ASCII to reveal the message

General Tips for Success

Start Simple

Get basic functionality working first, then add complexity. A working simple solution is better than a broken complex one!

Test Frequently

Run your code often to catch problems early. Test each new feature as you add it.

Use Comments

Explain your logic with comment blocks. This helps you remember your approach and shows your thinking.

Debug Systematically

When something doesn't work, isolate the problem. Test one section at a time.

Share Your Success

Document Your Work:
  • Take screenshots of your successful runs
  • Save your project files with descriptive names
  • Write a brief description of your approach
  • Share with your classmates and learn from each other!
