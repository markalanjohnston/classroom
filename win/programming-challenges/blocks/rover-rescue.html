Rover Rescue: Advanced Resource Management

Master survival, resource management, and strategic planning in an alien environment

Welcome to Rover Rescue!

The Rover Rescue playground is unlike any other VEXcode VR environment. Your VR Rover has crash-landed on an alien planet and must survive by managing resources, fighting off enemies, and collecting minerals to power up and level up.

VR Rover Specifications

Dimensions: 191mm × 147mm
Starting Position: (-6000, -3000) near Research Base
Special Tool: Multi-purpose claw for minerals
Battery: 100% (depletes over time)
Key Difference: Unlike other VEX VR playgrounds, Rover Rescue features:
  • Limited battery that depletes over time
  • XP and leveling system
  • Resource collection and management
  • Enemy encounters with radiation absorption
  • Respawning minerals in certain zones

Your Mission

Program your VR Rover to autonomously survive as long as possible while:

  • Collecting and managing minerals
  • Fighting off irradiated enemies
  • Maintaining battery levels
  • Gaining XP and leveling up
  • Transporting resources to the Research Base

The Rover Rescue Playground

The alien environment is a 12,000mm × 6,000mm area divided into distinct zones, each with unique challenges and opportunities.

Playground Layout

Total Area: 12,000mm × 6,000mm
Grid Spacing: 500mm between squares
Number of Zones: 5 (A through E)
Research Base: Southwest corner

Key Features

Research Base

Your home base for depositing minerals

Minerals

Energy sources scattered throughout

Enemies

Irradiated creatures to neutralize

Obstacles

Barriers to navigate around

Navigation Tip

The playground includes a hazardous green river that's impassable. Use the two bridges connecting Zone A's eastern bank and Zone B's northern bank to cross safely.

Resources & Experience System

Minerals: Your Lifeline

Minerals are essential for survival. They can be used to recharge your battery or transported to the base for XP.

Mineral Actions & Rewards

Pick Up: Store in cargo (limited capacity)
Use (on ground): +2 XP, Full battery recharge
Drop at Base: +5 XP per mineral

Battery Management

Battery:
100%

Your battery constantly depletes. When it reaches 0%, your mission ends. Use minerals or enter standby mode to manage battery life.

Experience & Leveling

Gain XP to level up and prove your programming skills:

  • Use minerals: +2 XP
  • Deliver to base: +5 XP per mineral
  • Neutralize enemies: +5 to +15 XP (depends on enemy)

Efficiency Tip

Delivering minerals to base gives more XP than using them directly, but using them provides instant battery recharge. Balance immediate needs with long-term gains!

Enemies & Combat System

Irradiated creatures roam the alien landscape. Neutralize them by absorbing their radiation!

Combat Mechanics

Basic Combat Strategy:
1. Detect enemy with Rover detects block
2. Approach enemy with Go to object block
3. Use Absorb enemy radiation block
4. Repeat absorption based on enemy level
5. Gain XP when enemy is neutralized
Combat Tip: Higher-level enemies require multiple radiation absorptions. Check enemy level and radiation before engaging!

Rover Rescue Special Blocks

These unique blocks are only available in the Rover Rescue playground:

Action Blocks

[Go to object] - Navigate to minerals, enemies, or base
[Minerals action] - Pick up, use, or drop minerals
[Absorb enemy radiation] - Neutralize enemies
[Standby] - Enter low-power mode until battery threshold

Sensing Blocks

<Rover sees> - Detect objects in 40° field of view
<Rover detects> - Detect within 800mm radius
<Rover direction> - Get angle to objects (-180 to 180°)
<Rover distance> - Get distance to objects
<Rover location> - Get X/Y coordinates of objects

Status Blocks

<Minerals in storage> - Current mineral count
<Storage capacity> - Maximum minerals you can carry
<Battery level> - Current battery percentage
<Level> - Your current level
<XP> - Your experience points
<Under attack> - True when being attacked
<Enemy level> - Level of nearest enemy
<Enemy radiation> - Radiation of nearest enemy

Event Blocks

[When under attack] - Trigger when enemy attacks
[When level up] - Trigger when gaining a level

Survival Strategies

Success in Rover Rescue requires careful planning and adaptive strategies.

Basic Survival Loop

1. Resource Collection Phase

  1. Check battery level
  2. If low, find and use nearest mineral
  3. If adequate, collect minerals for base delivery
  4. Transport to base when cargo is full

2. Combat Phase

  1. Detect enemies in range
  2. Evaluate enemy level vs. battery status
  3. Engage if safe, retreat if risky
  4. Absorb radiation for XP

3. Zone Progression

  1. Start in Zones A/B for initial resources
  2. Clear easy enemies for XP
  3. Move to Zone C when minerals depleted
  4. Progress to D/E as you level up

Advanced Techniques

Standby Strategy: Use standby mode strategically! It slows battery drain and increases "Days Survived" faster. Set a threshold (like 35%) to exit standby and resume activities.
Base Proximity: The Rover always knows the base direction and distance. Use this for emergency returns when battery is critical!
Enemy Farming: In renewable zones, balance mineral collection with enemy hunting. Both respawn, providing unlimited XP opportunities!

Starter Template

[When started]
  [Forever]
    [If then else] <Battery level> < 30
      // Emergency: Find and use mineral
      [If then] <Rover detects [minerals]>
        [Go to object [minerals]]
        [Minerals action [use]]
    [else]
      // Normal operation
      [If then] <Minerals in storage> = <Storage capacity>
        // Cargo full, return to base
        [Go to object [base]]
        [Minerals action [drop]]
      [else]
        // Continue collecting
        [If then] <Rover detects [minerals]>
          [Go to object [minerals]]
          [Minerals action [pick up]]

[When under attack]
  [Absorb enemy radiation]
  [If then] <Enemy radiation> > 0
    [Repeat] <Enemy level>
      [Absorb enemy radiation]
