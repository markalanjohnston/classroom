Getting Started with VEX VR

Welcome to VEXcode VR! In this lesson, you'll learn everything you need to know to start programming virtual robots.

No Downloads Required! VEXcode VR runs entirely in your web browser. All you need is an internet connection.

Launching VEXcode VR

Getting started with VEXcode VR is simple:

  1. Open your web browser (Chrome, Firefox, Safari, or Edge)
  2. Navigate to vr.vex.com
  3. When the welcome screen appears, click "Students Get Started"
  4. You're ready to code!

The VEXcode VR welcome screen

Pro Tip: Bookmark vr.vex.com for quick access to VEXcode VR in future sessions!

Understanding the Interface

VEXcode VR has four main areas you'll work with:

Toolbox

Contains all the blocks organized by category (Motion, Looks, Events, etc.)

Workspace

Where you build your programs by dragging and connecting blocks

Playground

Virtual environment where your robot runs and interacts

Toolbar

Project name, save/load options, and settings

The VR Robot

Your virtual robot comes pre-configured with:

  • Drivetrain: For movement (forward, backward, turning)
  • Pen: For drawing on the playground
  • Sensors: Bumper, distance, color, and location sensors
  • Electromagnet: For picking up disks

Understanding Block Shapes

VEXcode VR uses five different block shapes, each with a specific purpose:

The five types of blocks in VEXcode VR

Hat Blocks

Start your program (curved top). Example: "when started"

Stack Blocks

Perform actions (rectangular). Example: "drive forward"

Boolean Blocks

Return true/false (hexagonal). Example: "bumper pressed?"

Reporter Blocks

Report values (rounded). Example: "distance in mm"

C Blocks

Control flow (C-shaped). Example: "repeat" or "if then"

Shape Matters! Blocks only fit where their shapes match. This prevents coding errors!

Your First Project

Let's create a simple program to make the robot move and draw a square:

  1. Drag a "when started" block from Events to the workspace
  2. From the Pen category, add "pen down"
  3. From Control, add a "repeat 4" block
  4. Inside the repeat block, add:
    • "drive forward for 200 mm" (from Drivetrain)
    • "turn right for 90 degrees" (from Drivetrain)
  5. Click the Playground button to see your robot
  6. Click Start to run your program!

Tips & Tricks

Working with Blocks

  • Delete blocks: Drag them back to the toolbox or right-click and select delete
  • Duplicate blocks: Right-click and select "Duplicate"
  • Get help: Click the question mark on any block to learn more
  • Undo/Redo: Use Ctrl+Z (Cmd+Z on Mac) and Ctrl+Y (Cmd+Y on Mac)

Saving Your Work

  1. Click on the project name at the top (default: "VEXcode Project")
  2. Type a new name for your project
  3. Go to File menu → "Save to Your Device"
  4. Your project downloads to your computer
Important: VEXcode VR doesn't auto-save. Remember to save your work before closing the browser!

Switch Blocks (Advanced)

As you advance, you can use Switch blocks to transition from visual blocks to Python code:

Switch blocks show Python code inside visual blocks

Not ready for Switch blocks? No problem! Master regular blocks first, then explore Switch blocks when you're ready for Python.
