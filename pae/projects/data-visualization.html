Data Visualization Practice: Kaggle + Pandas + Matplotlib

Self-Paced · 20 min Turn in a screenshot on Schoology

Learning Goal

Use real-world data in Google Colab to create a professional chart with:

  • Real dataset loaded from Kaggle
  • Pandas for data manipulation
  • Matplotlib for creating visualizations
  • Custom colors, markers, and styling
  • Professional labels and titles

Follow These Steps

  1. Create a new Google Colab notebook

    Go to colab.research.google.com and click + New Notebook

    Rename your notebook to "Data_Visualization_Practice"

  2. Download the dataset

    Click this link to download the CSV file (it should download automatically):

    World-population-by-countries-dataset.csv

    Save it to your Downloads folder. If it opens in your browser instead, right-click on the page and select "Save As..."

  3. Upload the dataset to Colab

    In your Colab notebook, run this code to upload the CSV file:

    from google.colab import files
uploaded = files.upload()

    Click the Choose Files button and select your downloaded CSV file.

    Tip: The file will upload to Colab's temporary storage. It will stay there for your entire session!

  4. Import libraries and load the data

    Pandas and Matplotlib are already installed in Colab! Just import them and load your data:

    import pandas as pd
import matplotlib.pyplot as plt

# Load the CSV file into a DataFrame
df = pd.read_csv("World-population-by-countries-dataset.csv")

# Check what columns we have
print(df.columns)

# Preview the first 5 rows
df.head()

    Run this cell. Look at the column names that print out - we'll use those in the next step!

  5. Create a basic bar chart

    First, let's see what columns we actually have. The dataset might have 'Year' and 'Value' columns OR year columns like '2022'. Check your columns first:

    # IMPORTANT: Check your actual column names!
print("Columns in your dataset:")
print(df.columns.tolist())
print("\nFirst few rows:")
print(df.head())

    Option A: If you see columns like 'Country Name', 'Year', 'Value':

    # Filter for most recent year
latest_data = df[df['Year'] == df['Year'].max()]
top10 = latest_data.nlargest(10, 'Value')

plt.figure(figsize=(10, 6))
plt.bar(top10['Country Name'], top10['Value'])
plt.title(f"Top 10 Countries by Population ({df['Year'].max()})")
plt.xlabel("Country")
plt.ylabel("Population")
plt.xticks(rotation=45, ha='right')
plt.tight_layout()
plt.show()

    Option B: If you see columns like 'Country/Territory' and year columns (numbers):

    # Find the last numeric column (most recent year)
year_cols = [col for col in df.columns if col.replace('.','').isdigit()]
if year_cols:
    latest_year = year_cols[-1]
    top10 = df.nlargest(10, latest_year)

    plt.figure(figsize=(10, 6))
    plt.bar(top10.iloc[:, 0], top10[latest_year])  # First column is country
    plt.title(f"Top 10 Countries by Population")
    plt.xlabel("Country")
    plt.ylabel("Population")
    plt.xticks(rotation=45, ha='right')
    plt.tight_layout()
    plt.show()
else:
    print("Could not find year columns!")
  6. Customize your chart with colors and styles (REQUIRED for full points)

    Now make your chart look professional with custom styling. Use the same approach as Step 5, but add customizations:

    For Option A data (Year/Value columns):

    # Get latest year data
latest_data = df[df['Year'] == df['Year'].max()]
top10 = latest_data.nlargest(10, 'Value')

# Create a customized chart
plt.figure(figsize=(12, 6), facecolor="lightyellow")  # Background color
plt.bar(
    top10['Country Name'],
    top10['Value'],
    color='purple',           # Custom bar color
    edgecolor='darkblue',     # Border color
    linewidth=2               # Border thickness
)

plt.title(f"Top 10 Countries by Population ({df['Year'].max()})", fontsize=18, color="darkblue", fontweight='bold')
plt.xlabel("Country", fontsize=14)
plt.ylabel("Population", fontsize=14)
plt.xticks(rotation=45, ha='right')
plt.grid(True, axis='y', linestyle="--", alpha=0.5)  # Add horizontal grid
plt.tight_layout()
plt.show()

    For Option B data (year columns):

    # Use the most recent year column
year_cols = [col for col in df.columns if col.replace('.','').isdigit()]
latest_year = year_cols[-1] if year_cols else None

if latest_year:
    top10 = df.nlargest(10, latest_year)

    plt.figure(figsize=(12, 6), facecolor="lightyellow")
    plt.bar(
        top10.iloc[:, 0],  # Country column
        top10[latest_year],
        color='purple',
        edgecolor='darkblue',
        linewidth=2
    )

    plt.title(f"Top 10 Countries by Population", fontsize=18, color="darkblue", fontweight='bold')
    plt.xlabel("Country", fontsize=14)
    plt.ylabel("Population", fontsize=14)
    plt.xticks(rotation=45, ha='right')
    plt.grid(True, axis='y', linestyle="--", alpha=0.5)
    plt.tight_layout()
    plt.show()

    Customization Ideas: Try changing color to "red", "green", "orange", "teal", etc. Change facecolor to "lightblue" or "lightpink". Try different edgecolor options!

Troubleshooting Common Issues

Problem: File not found error

Solution: Make sure you uploaded the file using files.upload() and the filename in your code matches exactly. Check the filename:

# List uploaded files
import os
print(os.listdir())

Then use the exact filename you see in the list.

Problem: Column not found error (KeyError)

Solution: The column names must match exactly. Check your column names:

# Check your column names
print(df.columns)

Make sure you're using 'Country/Territory' for the country name and '2022' (or whatever year) for the population data.

Problem: Chart labels are overlapping

Solution: Add rotation to your labels and use plt.tight_layout():

plt.xticks(rotation=45, ha='right')
plt.tight_layout()

Problem: Chart doesn't look good

Solution: Experiment with different figure sizes and colors:

plt.figure(figsize=(12, 6))  # Make it wider
plt.bar(..., color='steelblue')  # Try different colors
