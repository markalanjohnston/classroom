Step 1 — Upload the file to Colab

What's happening here: Google Colab runs in the cloud, so your files on your computer aren't automatically available. This code creates a button that lets you browse your computer and select a file to upload into Colab's temporary workspace.

The code breakdown:

  • from google.colab import files — This imports Google Colab's special file handling tools
  • files.upload() — This opens a file browser window on your screen
from google.colab import files
files.upload()

Action: When you run this code, click the "Choose Files" button that appears, then select population.csv from your Downloads folder.

Note: Files uploaded to Colab are temporary! They'll disappear when you close your Colab session, so you'll need to re-upload if you come back later.

Step 2 — Load the data

What's happening here: Now that your file is uploaded, we need to load it into Python's memory using specialized tools that can handle large datasets.

Understanding the libraries:

  • Pandas — A powerful Python library for working with data. Think of it like Excel, but for code! We nickname it pd to save typing.
  • Matplotlib — A library for creating charts and graphs. We import pyplot (the plotting part) and call it plt for short.

The code breakdown:

  • import pandas as pd — Loads the Pandas library and gives it the shorthand name "pd"
  • import matplotlib.pyplot as plt — Loads the charting tools and calls it "plt"
  • df = pd.read_csv("population.csv") — This is the magic! It reads your CSV file and creates a DataFrame (stored in the variable df)
  • df.head() — Shows the first 5 rows of your data so you can see it loaded correctly
What's a DataFrame? A DataFrame (df for short) is like a spreadsheet inside Python. It has rows and columns, and you can do calculations, sort, filter, and visualize the data. The variable name df is a common convention in data science — you'll see it everywhere! 
import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("population.csv")
df.head()  # Shows the first few rows so we know it loaded

What you'll see: After running this, you should see a table showing the first 5 rows of population data with countries and years as columns.

Step 3 — Prepare the data for graphing

What's happening here: Our CSV file has a column called "Value" that contains the population numbers. To make our code clearer, we'll rename it to "Population".

The code breakdown:

  • df.rename(...) — Changes the "Value" column name to "Population" for clarity
  • data_long = df.copy() — Creates a working copy of our data
  • data_long.head() — Shows the first 5 rows to verify everything looks correct
# Rename 'Value' column to 'Population' for clarity
data_long = df.rename(columns={'Value': 'Population'})

# View the data
data_long.head()

What you'll see: A table with four columns: Country Name, Country Code, Year, and Population. Each row represents one country in one specific year!

Step 4 — Plot a country

What's happening here: Now that our data is in the right format, let's create our first chart by selecting one country and plotting its population over time.

The code breakdown:

  • country = "United States" — Choose which country to graph. Note: This is just an example! You must change this to a different country for full credit.
  • data_long[data_long["Country Name"] == country] — Filters the data to only include rows for your chosen country
  • plt.figure(figsize=(12,6)) — Creates a blank chart that's 12 inches wide and 6 inches tall
  • plt.plot(...) — Draws a line connecting all the data points
    • data["Year"] — X-axis values
    • data["Population"] — Y-axis values
    • marker="o" — Puts a circle at each data point
    • linewidth=2 — Makes the line 2 pixels thick
  • plt.title(...) — Adds a title at the top
  • plt.xlabel(...) and plt.ylabel(...) — Labels for the axes
  • ax.yaxis.set_major_formatter(...) — Formats large numbers with commas (like 350,000,000 instead of 3.5e8)
  • plt.grid(True) — Adds helpful gridlines
  • plt.show() — Displays the finished chart!
country = "United States"  # Change country if you want

# Filter rows only for our country
data = data_long[data_long["Country Name"] == country]

import matplotlib.ticker as ticker

plt.figure(figsize=(12,6))
plt.plot(data["Year"], data["Population"], marker="o", linewidth=2)

plt.title(f"Population of {country}")
plt.xlabel("Year")
plt.ylabel("Population")

# Format numbers so they look normal instead of scientific notation
ax = plt.gca()
ax.yaxis.set_major_formatter(ticker.StrMethodFormatter('{x:,.0f}'))

plt.grid(True)
plt.show()

What you'll see: A blue line chart showing how your chosen country's population changed over time. Each point represents one year!

Step 5 — Style your graph (Required)

What's happening here: Now let's make your chart unique! Data visualization isn't just about showing data — it's about making it clear, engaging, and visually appealing.

Why customize? Professional charts stand out because of thoughtful design choices. Colors can emphasize trends, markers make data points easier to see, and styling can match your presentation theme or brand.

The code breakdown:

  • facecolor="lightyellow" — Sets the background color of the entire chart area (try "lightblue", "lavender", "mintcream", etc.)
  • color="purple" — Changes the line color from default blue to purple (try "red", "green", "orange", "#FF5733", etc.)
  • marker="s" — Changes the data point markers to squares:
    • "o" = circle (default)
    • "s" = square
    • "^" = triangle pointing up
    • "D" = diamond
    • "*" = star
    • "x" = x marks
  • linewidth=3 — Makes the line thicker (default is 2) so it's more prominent
  • color="darkblue" in title — Changes just the title text color to stand out
Your turn to experiment! Change the colors and markers to create your own unique style. This customization is required for full credit (10 points) — don't leave it as the default colors!

Color options:

  • Named colors: "red", "blue", "green", "purple", "orange", "pink", "brown", "black", etc.
  • Hex codes: "#FF5733", "#3498DB", "#2ECC71" (search "color picker" online to find hex codes)
import matplotlib.ticker as ticker

plt.figure(figsize=(12,6), facecolor="lightyellow")
plt.plot(
    data["Year"], data["Population"],
    color="purple", marker="s", linewidth=3
)

plt.title(f"Population of {country}", color="darkblue")
plt.xlabel("Year")
plt.ylabel("Population")

ax = plt.gca()
ax.yaxis.set_major_formatter(ticker.StrMethodFormatter('{x:,.0f}'))

plt.grid(True)
plt.show()

Challenge yourself: Try different combinations! Maybe a red line with star markers on a light blue background? Get creative and make it your own!
