Data Visualization Practice: Kaggle + Pandas + Matplotlib

Self-Paced · 20 min Turn in a screenshot on Schoology

Learning Goal

Use real-world data in Google Colab to create a chart with:

  • Data downloaded from Kaggle
  • Pandas for data loading
  • Matplotlib for plotting
  • Custom colors and axis labels

✅ Success Checklist (Match the Rubric)

  • I downloaded a real dataset
  • I loaded it into Pandas and previewed it
  • I made a chart using Matplotlib
  • I changed the line color and one more style (marker, background, grid, title color, etc.)
  • I labeled both axes and added a title
  • I took a screenshot showing my chart

Follow These Steps

  1. Open Colab

    Go to colab.research.google.com

  2. Install Kaggle tool 
    !pip install kaggle
  3. Upload Kaggle API key

    Download from Kaggle → Settings → Create API Token

    from google.colab import files
files.upload()
    !mkdir -p ~/.kaggle
!cp kaggle.json ~/.kaggle/
!chmod 600 ~/.kaggle/kaggle.json

    Note: If this doesn't work, scroll to the "Manual Upload Backup" below.

  4. Download dataset 
    !kaggle datasets download -d tanmoyxd/world-population-19602021
!unzip -o world-population-19602021.zip
  5. Load and preview data 
    import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("world_population.csv")
df.head()
  6. Plot one country's population 
    country = "United States"

data = df[df["Country"] == country]

plt.figure()
plt.plot(data["Year"], data["Population"], marker="o", linewidth=2)
plt.title(f"Population of {country}")
plt.xlabel("Year")
plt.ylabel("Population")
plt.show()
  7. Customize your chart (Required) 
    plt.figure(facecolor="lightyellow")
plt.plot(
    data["Year"], data["Population"],
    color="purple", marker="s", linewidth=3
)

plt.title(f"Population of {country}", color="darkblue")
plt.xlabel("Year")
plt.ylabel("Population")
plt.grid(True)
plt.show()
  8. Take a screenshot & submit to Schoology

    Make sure screenshot shows: your chart, custom color, axis labels, title
