Step 2 — Load the data

This imports Pandas (data library) and Matplotlib (chart library), then reads your file.

import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("population.csv")
df.head()  # Shows the first few rows so we know it loaded

Step 3 — Reshape the data + plot a country

The dataset is originally in a "wide" format: each year is its own column.
We need to "melt" it into a "long" format with 3 columns: Country, Year, Population. This makes it easier to graph. 

country = "United States"  # Change country if you want

# Reshape from wide format to long format
data_long = df.melt(id_vars=["Country Name", "Country Code"],
                    var_name="Year",
                    value_name="Population")

# Convert Year text → number
data_long["Year"] = pd.to_numeric(data_long["Year"], errors="coerce")

# Filter rows only for our country
data = data_long[data_long["Country Name"] == country]

import matplotlib.ticker as ticker

plt.figure(figsize=(12,6))
plt.plot(data["Year"], data["Population"], marker="o", linewidth=2)

plt.title(f"Population of {country}")
plt.xlabel("Year")
plt.ylabel("Population")

# Format numbers so they look normal instead of scientific notation
ax = plt.gca()
ax.yaxis.set_major_formatter(ticker.StrMethodFormatter('{x:,.0f}'))

plt.grid(True)
plt.show()

Step 4 — Style your graph (Required)

Customize background, color, and marker.

import matplotlib.ticker as ticker

plt.figure(figsize=(12,6), facecolor="lightyellow")
plt.plot(
    data["Year"], data["Population"],
    color="purple", marker="s", linewidth=3
)

plt.title(f"Population of {country}", color="darkblue")
plt.xlabel("Year")
plt.ylabel("Population")

ax = plt.gca()
ax.yaxis.set_major_formatter(ticker.StrMethodFormatter('{x:,.0f}'))

plt.grid(True)
plt.show()
