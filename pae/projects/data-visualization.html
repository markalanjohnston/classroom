Data Visualization Practice: Kaggle + Pandas + Matplotlib

Self-Paced · 20 min Turn in a screenshot on Schoology

Learning Goal

Use real-world data in Google Colab to create a professional chart with:

  • Real dataset loaded from Kaggle
  • Pandas for data manipulation
  • Matplotlib for creating visualizations
  • Custom colors, markers, and styling
  • Professional labels and titles

✅ Success Checklist (Match the Rubric)

  • I downloaded a real dataset from Kaggle using kagglehub
  • I loaded it into a Pandas DataFrame and previewed it with df.head()
  • I created a line chart using Matplotlib showing year vs population
  • I customized the line color AND added at least one more style (marker, grid, background, or title color)
  • I added clear labels for both X and Y axes
  • I added a descriptive chart title
  • I took a screenshot showing my final customized chart

Follow These Steps

  1. Create a new Google Colab notebook

    Go to colab.research.google.com and click + New Notebook

    Rename your notebook to "Data_Visualization_Practice"

  2. Download the dataset from Kaggle

    Use kagglehub to download the dataset directly in Colab (no manual download needed!):

    import kagglehub

# Download latest version
path = kagglehub.dataset_download("kaggleashwin/population-dataset")

print("Path to dataset files:", path)

    Run this cell. You'll see the path where the dataset was downloaded.

    Cool! Kagglehub automatically downloads the dataset for you - no need to manually download and upload files!

  3. Import libraries and load the data

    Now load the data with Pandas (already installed in Colab!):

    import pandas as pd
import matplotlib.pyplot as plt

# Load the CSV file into a DataFrame
df = pd.read_csv(path + "/population_by_country.csv")

# Preview the first 5 rows
df.head()

    Run this cell. You should see a table showing the first 5 rows of data with columns like Country, Year, and Population.

  4. Create a basic chart for one country

    Let's plot the population of the United States over time:

    # Choose a country
country = "United States"

# Filter data for just this country
country_data = df[df["Country"] == country]

# Create the plot
plt.figure(figsize=(10, 6))
plt.plot(country_data["Year"], country_data["Population"])
plt.title(f"Population of {country}")
plt.xlabel("Year")
plt.ylabel("Population")
plt.show()

    You should see a basic line chart showing population growth!

  5. Customize your chart with colors and styles (REQUIRED for full points)

    Now make your chart look professional with custom styling:

    # Choose your country
country = "United States"

# Filter the data
country_data = df[df["Country"] == country]

# Create a customized chart
plt.figure(figsize=(10, 6), facecolor="lightyellow")
plt.plot(
    country_data["Year"],
    country_data["Population"],
    color="purple",           # Custom line color
    marker="o",               # Add circle markers
    linewidth=2.5,            # Thicker line
    markersize=5              # Marker size
)

plt.title(f"Population Growth: {country}", fontsize=16, color="darkblue")
plt.xlabel("Year", fontsize=12)
plt.ylabel("Population", fontsize=12)
plt.grid(True, linestyle="--", alpha=0.5)  # Add grid
plt.tight_layout()
plt.show()

    Customization Ideas: Try changing color to "red", "green", "orange", etc. Change marker to "s" (square), "^" (triangle), or "D" (diamond). Change facecolor to "lightblue" or "lightgray".

  6. Try different countries (Optional but fun!)

    Change the country variable to compare different countries:

    # Try these countries:
# "China", "India", "Brazil", "Mexico", "Japan", "Germany"

country = "China"  # Change this!
country_data = df[df["Country"] == country]

plt.figure(figsize=(10, 6))
plt.plot(country_data["Year"], country_data["Population"],
         color="red", marker="s", linewidth=2.5)
plt.title(f"Population Growth: {country}", fontsize=16)
plt.xlabel("Year", fontsize=12)
plt.ylabel("Population", fontsize=12)
plt.grid(True)
plt.show()
  7. Take a screenshot and submit to Schoology

    Make sure your screenshot clearly shows:

    • Your customized chart with a custom color
    • At least one additional style (marker, grid, background color, or colored title)
    • Clear axis labels (X and Y)
    • A descriptive title

    How to screenshot:

    • Chromebook: Press Ctrl + Show Windows key (or Ctrl + F5), then click and drag
    • Windows: Press Windows + Shift + S
    • Mac: Press Cmd + Shift + 4, then drag to select

Troubleshooting Common Issues

Problem: File not found error

Solution: Make sure the path variable is correct. Try printing it first:

print(path)
print(path + "/population_by_country.csv")

If the file has a different name, check what files are in the path:

import os
print(os.listdir(path))

Problem: Column not found error

Solution: Run df.columns to see the exact column names in your dataset. The column names must match exactly (including capitalization).

# Check your column names
print(df.columns)

Problem: Chart doesn't show my country

Solution: Check the spelling and capitalization. Run this code to see all available countries:

# See all available countries
print(df["Country"].unique())

Want to use a different dataset?

You can! Search for datasets on Kaggle and use kagglehub to download them. Just make sure the dataset has at least two columns of numeric data (like Year and Population, or Date and Temperature).
