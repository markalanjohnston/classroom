Step 5 ‚Äî Plot All 4 Lines

This loop repeats the plotting code for each country automatically.

import matplotlib.ticker as ticker

plt.figure(figsize=(12,7), facecolor="white")

for country in countries:
    data = data_long[data_long["Country Name"] == country]

    if data.empty:
        print(f"‚ö†Ô∏è '{country}' not found ‚Äî check spelling.")
        continue

    plt.plot(
        data["Year"],
        data["Population"],
        marker="o",
        linewidth=2,
        label=country
    )

plt.title("Population Comparison: 4 Countries", color="darkblue")
plt.xlabel("Year")
plt.ylabel("Population")

plt.legend()
plt.grid(True, alpha=0.3)

ax = plt.gca()
ax.yaxis.set_major_formatter(ticker.StrMethodFormatter('{x:,.0f}'))

plt.show()

Optional Challenge ‚Äî Compare % Growth Instead of Size

When you compare a huge country (like China) to a smaller one (like Germany), the smaller country‚Äôs line can look flat or ‚Äúzero‚Äù ‚Äî even when it‚Äôs changing!

To fix this, we can graph percentage change instead of total population. This shows how fast each country grows, not just how big it is.

Idea: We start each country at 100%. If a population rises to 120%, that means it grew 20% from the first year.
üìä Code ‚Äî Plot Percent Change
import matplotlib.ticker as ticker

plt.figure(figsize=(12,7), facecolor="white")

for country in countries:
    data = data_long[data_long["Country Name"] == country].copy()

    if data.empty:
        print(f"‚ö†Ô∏è '{country}' not found ‚Äî check spelling.")
        continue

    # Calculate percent change relative to first year
    start_value = data["Population"].iloc[0]
    data["Percent"] = (data["Population"] / start_value) * 100

    plt.plot(
        data["Year"],
        data["Percent"],
        marker="o",
        linewidth=2,
        label=country
    )

plt.title("Population Change Over Time (Percent Growth)", color="darkblue")
plt.xlabel("Year")
plt.ylabel("Population (% of starting value)")

plt.legend()
plt.grid(True, alpha=0.3)

ax = plt.gca()
ax.yaxis.set_major_formatter(ticker.StrMethodFormatter('{x:,.0f}%'))

plt.show()
‚úÖ Now you can see who is growing faster ‚Äî not just who has the most people.
‚ö†Ô∏è This doesn't show total population ‚Äî it shows growth trend.
