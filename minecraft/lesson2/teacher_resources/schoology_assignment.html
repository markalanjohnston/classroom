Assignment — Minecraft Lesson 2

Loops, Repetition, and Agent Patterns

The Task

Create an original Agent program that uses at least one for loop to build something in Minecraft — a path, a wall, a square, a tower, or any pattern you design. Your final program must be different from the starter examples in the lesson.

Minimum requirements:

  • Your program starts with a chat command.
  • Your program uses at least one for loop.
  • The Agent places at least 10 blocks total.
  • You changed the loop count at least once and can describe what changed.
  • The final program is different from the starter examples on the lesson page.

What to Turn In

Submit one PDF file to this assignment. Build it in Microsoft Word or Google Docs, then export it as a PDF before uploading.

Your document must include all three of the following, in this order:

  1. Screenshot of your code in the MakeCode editor (pasted into the document). Your for loop must be clearly visible.
  2. Screenshot of the result in Minecraft (pasted into the document) — the path, wall, square, or pattern your Agent built.
  3. Short written reflection — type your answers to all four questions below directly into the same document (1–2 sentences each).

How to save it as a PDF:

  • Microsoft Word: File → Save As → choose PDF from the file-type dropdown.
  • Google Docs: File → Download → PDF Document (.pdf).

Upload only the PDF — do not submit the Word doc, the Google Docs link, or the raw image files.

Reflection Questions

Answer all four. Aim for 1–2 sentences each — show that you understand why the loop worked, not just that it worked.

  1. What does your loop repeat, and how many times?
  2. Which number did you change to make the result bigger or smaller? What happened?
  3. Describe one bug you ran into — what was wrong, and how did you figure it out?
  4. How would this program look if you wrote it without a loop?

Tips for a Strong Submission

  • Use the full-screen screenshot tool so your code is easy to read.
  • If your build is hard to see from one angle, fly up (creative mode) and take a top-down shot.
  • In your reflection, name the exact line you changed and what number you changed it from and to.
  • If you used a nested loop or multiple block types, mention it — that's stretch-goal territory.
