Teacher Tools

Small utilities I use for day-to-day classroom work.

About this page

These are lightweight, browser-based tools I've built for my own classroom. They run entirely in the browser — nothing is sent to a server, and no account is needed. Feel free to use them if they help you too.

Available Tools

Thermo

Thermal Printer

Prints hall passes, test-print receipts, and other small classroom slips to an 80mm thermal printer. Includes live preview at true paper width, multiple templates, and QR code support.
